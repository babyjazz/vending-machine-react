# Vending machine playgrond

## requirement
- node V16
- yarn
- git

##  Run development
```
yarn install
yarn start
```
Server will run on `localhost:3000` and connect to api at endpoint `localhost:3001`
You can check endpoint from ENV

## Run production
```
yarn build
```
Production will built in build folder
